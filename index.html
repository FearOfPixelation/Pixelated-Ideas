How to make HTML websites with ChatGPT

Website design has never been easier, here's how to make a website with ChatGPT! Firstly, go to the ChatGPT website. Now, ask ChatGPT to create "HTML code" with your website idea in the same message.

Now, copy the code. Open notepad and paste it into it.

Now, hover over files, then click save as

Select save as type, do all files.

Name it .html at the end, then click save.

You now have a website file. You can open it to run it on your browser locally. But to get it online you now need a domain. You can either get a domain with GitHub (How this website is running) or you can get paid domains. Here are some domain hosters:

What is temperature in AI terms

In various advanced AI applications, there is a setting called temperature that can be seen.

What is temperature. To put it simply, it decides how random the system is. If a temperature is 0, the AI will be deterministic. It will always spit out the same message if given the same prompt in different chats. These prompts will usually lack creativity, but it will be more professional and coherent.

The higher a temperature is the more random it gets, causing more creativity. A moderate temperature is usually .7. If a temperature is set too high, it may start generating absurd prompts with near to no cohesion.
